# Landslide-Risk-Prediction
This project presents an automated geospatial landslide risk assessment system built using Google Earth Engine (GEE). It integrates terrain, vegetation, and rainfall data with a Frequency Ratio (FR) model to dynamically estimate landslide susceptibility and generate near real time risk alerts at the district level in Bangladesh.


Launch the App: [Smart Geospatial Landslide Risk Prediction & Alert System](https://pritomboseaapon.users.earthengine.app/view/smart-geospatial-landslide-risk-prediction--alert-system)
