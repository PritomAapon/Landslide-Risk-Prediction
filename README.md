# Landslide-Risk-Prediction
This project presents an automated geospatial landslide risk assessment system built using Google Earth Engine (GEE).


This repository presents an integrated Frequency Ratio (FR)–Multi-Criteria Decision Analysis (MCDA) framework for automated landslide susceptibility mapping and near–real-time risk monitoring in the Chittagong Hill Tracts (CHT) of Bangladesh. Implemented in Google Earth Engine (GEE), the model combines terrain (SRTM), vegetation (Sentinel-2 NDVI), and rainfall (CHIRPS) data to generate a normalized Landslide Susceptibility Index (LSI). Dynamic hazard assessment is achieved by integrating short-term cumulative rainfall with the susceptibility layer, producing a multi-level risk surface (No Warning, Caution, Alert, Alarm). An interactive GEE dashboard allows users to generate location- and date-specific risk maps and warnings, providing a scalable, reproducible system for operational early warning in high-risk, data-scarce regions.

All historical landslide records were obtained from the [NASA Global Landslide Catalog](https://data.nasa.gov/dataset/global-landslide-catalog-export). The dataset was filtered to extract events occurring in Bangladesh, resulting in a CSV file containing only Bangladeshi landslide data.
